
KBOT modified for cross-platform
NO Buildx, only build

usage:
| Command | Description |
|---------|-------------|
| `make build` | Build for current OS/architecture |
| `make linux` | Build for Linux (amd64) |
| `make arm` | Build for Linux (arm64) |
| `make macos` | Build for macOS (amd64) |
| `make windows` | Build for Windows (amd64) |
| `make all` | Build for all platforms |
| `make clean` | Remove build artifacts |
